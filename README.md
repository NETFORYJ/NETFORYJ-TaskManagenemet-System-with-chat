# TaskManagenemet-System-with-chat
CollabFlow is a full-stack task management and collaboration platform featuring real-time chat, file sharing, multi-tenant support, and JWT authentication. Built with Spring Boot, WebSocket (STOMP), MySQL, and Tailwind CSS.


CollabFlow is a comprehensive task management and team collaboration platform that enables real-time communication, file sharing, and task tracking. Built with Spring Boot and WebSocket technology, it provides instant updates for chat messages and task assignments, making it ideal for small to medium-sized teams.

Key Features:
✅ JWT Authentication with Google OAuth2 support
✅ Real-time Group & Private Chat (WebSocket/STOMP)
✅ Task Management (CRUD, Assign, Take, Complete)
✅ File Transfer Center (Upload/Download/Delete)
✅ Multi-tenant Architecture
✅ Dashboard with Analytics (Chart.js)
✅ Responsive UI with Tailwind CSS
✅ Audit Logging & Activity Tracking
✅ Online/Offline User Status
✅ Unread Message Indicators

Tech Stack: Spring Boot 3.2, Java 17, MySQL, WebSocket, JWT, Tailwind CSS, Chart.js
